chatGPT4.0升级教程

前言

最近，有很多中国用户在充值ChatGPT PLUS会员的时候，会收到如下提示：您的信用卡被拒绝了。请尝试使用借记卡支付。这可能是由于多种原因导致的，例如：信息填写错误，信用卡已过期、信用额度不足、卡片被冻结等等各种原因。

首先，需要确定的是，你是不是用的国内的信用卡？目前所有国内开户的信用卡都无法充值，不管是VISA还是万事达，所以如果你想要充值chatGPT，需要使用国外开户的信用卡。

这个教程教大家如何使用国外的虚拟信用升级去升级chatGPT4.0

关于虚拟信用卡

虚拟信用卡 (也叫 VCC) 除了解决 ChatGPT Plus 购买和绑定的问题外，还能解决众多国外购物平台（例如亚马逊）、网络服务/应用软件订阅、买游戏、购买 VPS 服务器等等的支付问题，甚至对那些绑定信用卡免费试用的活动还能薅薅羊毛。

而相比传统的实体银行信用卡，虚拟卡的好处是无需审核资质、注册方便快捷、无需等待实体卡，充值即用，而且可以批量多开卡，同时还有不同的卡组织 (VISA / Master / 运通等) 或卡号可以选择，能顺利通过很多用实体卡无法成功支付的场景，因此用途特别广泛。

虚拟信用卡平台 WildCard

在查阅及体验了一下众多的平台后，决定推荐一下 WildCard 这个平台。原因很简单，就是应为它简单快速，相较与其他平台的一通复杂繁琐的界面操作，简直就是小白以及不想浪费时间的同学们的福音。

开卡流程

  1. 首先我们点击进入: WildCard 官网

  2. 点击 立即体验 进行开卡操作

  3. 可以了解一下开卡价格，一年11.99美金，两年13.99美金，填邀请码能抵扣2美金

    也就是一天0.2人民币。这里提供一个邀请码给大家 GPTLEVE

  4. 剩下来的操作就是验证操作了

升级GPT4.0

  1. 登录 chat.openai.com 依次点击 Login ，输入邮箱和密码

    如果没有楼梯纸是上不去openai的网站的哦 这里挂一个平民且较稳定的 泡泡dog。

  2. 点击升级 Upgrade

    登录自己的 OpenAI 帐户后，点击左下角的 Upgrade to Plus，在弹窗中选择 Upgrade plan。

    如果升级入口无法点击，那就访问这个网址，https://chat.openai.com/invite/accepted

    进入支付页面。

  3. 复制支付页面网址

    在支付页面，复制现在的付款页网址 URL，这一段 pay.openai.com 开头的全部网址，进入我们「一键升级 ChatGPT Plus」页面，提交就可以了。请不要在支付页面填写任何信息，如果填写过，请返回上一步，重新进入这个页面再复制网址。

  4. 使用 WildCard 一键升级 ChatGPT Plus

    登录 wildcard， 点击一键升级。

    在输入框，粘贴刚刚复制的支付页面的网址，然后点击确定支付并升级。

    然后按步骤选择要支付的卡片，一路走下去就可以了。

到此就升级成功了
